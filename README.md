# vagrant-kubernetes-cluster

1- install virtual box <br/>
2- install vagrant  <br/>
3- clone this project and execute Vagrant file <br/>

  “nbsp” -> cd vagrant-kubernetes-cluster <br/>
  “nbsp” -> vagrant up <br/>
